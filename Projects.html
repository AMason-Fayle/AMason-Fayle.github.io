Battleships:

This project was built using Python 3 and its TKinter library,
around March 2021.


It is a mouse controlled game of Battleships.

Ships are placed by selecting the two endpoints,
the programme will validate this automatically to
ensure that only the appropriate number, and type,
of ships are placed.

Cistercian Numeral Display:

This project was built using Python 3 and its TKinter library,
around March 2021.


It is a display for the Cistercian Numeral system,
invented by Cistercian monks before the adoption
of Arabic numerals, for writing the years more
concisely than was possible with Roman numerals.


This display is styled after a 7-segment display.
